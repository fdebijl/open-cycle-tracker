## Categories

While users can adjust the categories they want to track in OCT, the default set of categories has to be a set that makes sense for most users, as to avoid onerous customization before OCT becomes usable. We've based this set on the work of Li et al. in *[Characterizing physiological and symptomatic variation in menstrual cycles using self-tracked mobile-health data](https://doi.org/10.1038/s41746-020-0269-8)* (2020). This article outlines the categories and symptoms available in the popular period tracking app Clue, and what percentage each category makes up as a fraction of the total number of events tracked. The set of categories included in OCT by default are categories that make up more than 1% of tracked events.
